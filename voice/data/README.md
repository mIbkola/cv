# Voice — Data

Хранилище датасетов для VOICE-модуля.

## Структура
```
data/
├── voice_samples/
│   ├── raw/            # исходники (mp3, wav, mp4) с голосом Бурунова
│   └── processed/      # нарезанные фразы 3–15 сек, 22050 Hz моно
├── jokes_corpus.jsonl  # база анекдотов 1980-х
└── README.md
```

## Источники
### Голос Бурунова
- Интервью на YouTube → выкачать `yt-dlp`
- Аудиокниги (если есть в исполнении Бурунова)
- Дубляж фильмов
- Минимум 10 минут чистого голоса

### Анекдоты 1986
- Открытые сборники анекдотов СССР
- Форумы / сетевые архивы 90-х
- Печатные сборники (если есть в цифре)

## Формат jokes_corpus.jsonl
```json
{"id": "j_0001", "year": 1986, "topic": "штирлиц", "text": "Штирлиц подошел к окну..." }
```
