# Voice — RAG (анекдоты 1986)

RAG-система: по запросу «расскажи анекдот из 86-го» / «анекдот про штирлица» находит релевантный анекдот в базе и подаёт его в TTS (голос Бурунова).

## Стек
- **Эмбеддинги (векторизаторы текста):** `sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2`
- **Векторная БД:** Qdrant (локально, через Docker)
- **LLM (опционально):** Qwen2.5-7B-Instruct (локально через vLLM) — для разбора интента и адаптации текста
- **ASR (распознавание речи):** Whisper — если вход голосовой

## Установка
```bash
cd voice/rag
pip install -r requirements.txt

# Запуск Qdrant (векторная БД)
docker run -d -p 6333:6333 qdrant/qdrant
```

## Структура данных
`voice/data/jokes_corpus.jsonl` — построчно:
```json
{"id": "j_0001", "year": 1986, "topic": "штирлиц", "text": "..."}
```

## Сборка индекса
```bash
python build_index.py \
  --corpus ../data/jokes_corpus.jsonl \
  --qdrant-url http://localhost:6333 \
  --collection jokes_1986
```

## Полный пайплайн
```bash
python pipeline.py --input "расскажи анекдот 86 года про штирлица"
```

Что происходит внутри:
1. ASR (если вход голосовой) → текст
2. LLM разбирает интент: `{action: "joke", topic: "штирлиц", year: 1986}`
3. RAG-поиск: топ-5 релевантных анекдотов
4. Переранжирование cross-encoder'ом → топ-1
5. Опционально: LLM-адаптация (например, добавить «а вы слышали...»)
6. TTS (голос Бурунова) → аудио → динамик
