# Unitree G1 EDU Ultimate D — Технические характеристики

## Основные характеристики

| Параметр | Значение |
|---|---|
| Тип | Гуманоидный робот |
| Назначение | Программно-аппаратный комплекс для разработок |
| Процессоры | 8-ядерный. Опционально — NVIDIA Jetson Orin |
| Тип движителя | Опоры |
| Количество движителей | 2 |
| Дополнительная поддержка | Не требуется |
| Варианты передвижения | Вперед, назад, вбок, повороты, прыжки, бег, поднятие после падения |
| Максимальная скорость | 2 м/с |
| Количество степеней свободы | 43 |

### Степени свободы (всего 43)
- **Ноги:** 6 на каждую (12)
- **Руки:** 5 на каждую (10)
- **Кисти:** 7 на каждую (14) — *захват RH56DFTP*
- **Торс:** 1 + 2 (дополнительные моторы)

## Сканирование окружения
- 3D камеры
- Камера глубины: **Intel RealSense D435**
- Лазерные дальномеры
- **3D Lidar Livox MID-360**

## Интерфейсы
### Беспроводные
- WiFi 6
- Bluetooth 5.2

### Проводные
- 3× USB 3.0 Type-C Host (5VDC / 1.5A)
- 1× USB 3.2 Type-C Host / DP 1.4
- 2× RJ45 1Gbps

### Управление
- Ручное (контроллер)

## Установленное оборудование
- 3D Lidar **Livox MID-360**
- Камера глубины **Intel RealSense D435**
- Два дополнительных мотора на торс
- Захваты-«кисти рук» **RH56DFTP** (×2):
  - 12 суставов
  - 6 степеней свободы
  - интегрированные сенсоры усилия
  - диапазон 10–2500 г

|---|---|

## SDK и разработка
- **Официальный SDK:** `unitree_sdk2` (C++) / `unitree_sdk2py` (Python)
- **Sport mode API:** высокоуровневое управление ходьбой (`SportClient`)
- **Low-level API:** прямое управление двигателями (для манипуляций)
- **Robotics SDK / ROS2 wrapper:** community поддерживается
- **Камеры:** `librealsense2`, `livox_ros_driver2`
