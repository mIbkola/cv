# Coffee — Оркестратор (LLM + конечный автомат)

Принимает команду «принеси кофе Олегу», разбирает её на подзадачи, координирует остальные модули (детекция, манипуляция, локомоция).

## Подход: гибрид LLM + конечный автомат

LLM (или простой парсер) извлекает интент и параметры:
- `action: "deliver_item"` — действие
- `item: "coffee"` — объект
- `target_person: "Олег"` — получатель

Дальше — детерминированный конечный автомат, вызывающий инструменты (tools) из других модулей.

## Стек
- Python 3.11+
- `vllm` — для локального LLM Qwen2.5-7B-Instruct
- `pydantic` — для схем данных
- `transitions` — библиотека конечных автоматов
- ZeroMQ — для связи с другими модулями (опционально)

## Установка
```bash
cd coffee/llm_orchestrator
pip install -r requirements.txt
```

## Конечный автомат
```
IDLE (ожидание)
  ↓ (команда "принеси кофе Олегу")
PARSE_CMD (разбор команды)
  ↓
FIND_CUP (CV: поиск стакана)
  ↓ (получили 3D-координаты)
APPROACH_CUP (лоокомоция: подход к стакану)
  ↓ (дошли, в 0.5 м от стакана)
GRASP (манипуляция: хватка с контролем силы)
  ↓
STABILITY_CHECK (манипуляция: проверка стабильности хвата)
  ├─ fail → RETRY (до 3 раз) → FAILED → сообщить голосом
  └─ ok ↓
FIND_TARGET (CV: поиск Олега по лицу)
  ↓
APPROACH_TARGET (лоокомоция: подход к Олегу)
  ↓ (в 0.7 м от Олега)
HANDOVER (манипуляция: передача стакана)
  ↓ (сила упала < 20 г → стакан принят)
RELEASE (отпускание)
  ↓
IDLE
```

## Инструменты (tools) для LLM
```python
tools = [
    "find_object(class_name) -> {x, y, z}",          # CV
    "navigate_to(x, y) -> bool",                      # локомоция
    "grasp_with_force_feedback(target_pose) -> bool", # манипуляция
    "verify_grasp() -> {stable: bool, force: float}", # манипуляция
    "find_person(name) -> {x, y, z}",                 # CV (по лицу)
    "handover() -> bool",                             # манипуляция
    "speak(text) -> None",                            # голосовой модуль
]
```

## Использование
```bash
# Режим скрипта (без LLM, жёсткая логика)
python state_machine.py --command "принеси кофе Олегу"

# Режим с LLM (локальный Qwen2.5-7B через vLLM)
python agent.py --command "принеси кофе Олегу" --llm qwen-local
```

## Интеграция с голосовым модулем (VOICE)
- Команда может прийти голосом → ASR → оркестратор
- Статусные сообщения: «Иду за кофе», «Не нашёл стакан», «Олег, держи» → через TTS Бурунова
