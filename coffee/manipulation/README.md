# Coffee — Manipulation (руки + force feedback)

Управление руками G1 для хвата стакана с кофе без раздавливания и расплескивания.

## Аппаратные возможности
- **Захваты:** RH56DFTP × 2 (12 суставов, 6 DoF, сенсоры усилия 10–2500 г)
- **Длина руки:** 0.45 м
- **Макс. нагрузка:** 3 кг (стакан с кофе ≈ 200–400 г — запас есть)

> ⚠️ Руки на текущем экземпляре **не родные**. Перед запуском — обязательная калибровка суставов и тарировка сенсоров силы.

## Стек
- `unitree_sdk2py` (low-level arm control)
- `pyserial` (если сенсоры RH56DFTP по serial)
- ROS2 (опционально, для TF-дерева)

## Установка
```bash
cd coffee/manipulation
pip install -r requirements.txt
```

## Калибровка перед запуском
```bash
python calibrate.py  # тарировка force sensors на эталонных весах
```
Без калибровки хватка стакана будет неточной.

## Основные скрипты
```bash
# Грубое наведение по 3D координатам от CV
python move_to.py --x 0.45 --y 0.10 --z 0.30

# Плавное сближение + force-guided grasp
python grasp_controller.py --target 0.45,0.10,0.30

# Проверка стабильности после хвата
python stability_check.py

# Передача стакана Олегу
python handover.py
```

## Параметры хвата (для бумажного стакана)
| Параметр | Значение |
|---|---|
| Скорость сближения (последние 10 см) | ≤ 5 см/с |
| Порог срабатывания силы | 30–80 г |
| Порог деформации стакана | ~150 г (не превышать) |
| Anti-crush threshold | 200 г → мгновенный разжим |
| Slip detection | Δforce > 30% за 200 мс |
