# Coffee — Локомоция (ходьба + навигация)

Дойти до стакана, потом до Олега, неся стакан с кофе без расплескивания.

## Аппаратные возможности
- G1 EDU Ultimate D: макс. скорость 2 м/с, 2 опоры, способность вставать после падения
- 3D Lidar Livox MID-360 (для карты и обхода препятствий)
- RealSense D435 (фронтальная камера глубины)

## Стек
- `unitree_sdk2py` → `SportClient` (высокоуровневое управление ходьбой)
- `livox_ros_driver2` (опционально, через ROS2)
- `slam_toolbox` / `cartographer` для построения карты (опционально)
- Простая локальная навигация: обход препятствий через VFH / DWA

## Установка
```bash
cd coffee/locomotion
pip install -r requirements.txt
```

## Использование
```bash
# Построить карту комнаты
python build_map.py --scan-time 30 --output map.pcd

# Навигация к точке
python navigator.py --target 1.5,2.0,0.0 --map map.pcd --speed 0.6

# Поза с грузом (компенсация покачивания)
python carry_pose.py --enable
```

## Параметры движения с грузом
| Параметр | Без груза | С кофе |
|---|---|---|
| Скорость | до 2 м/с | 0.5–0.8 м/с |
| Длина шага | стандарт | уменьшенная |
| Высота корпуса | стандарт | на 2–3 см ниже (для центра масс) |
| Компенсация наклона | нет | активная через IMU |

## Фазы
1. **Сканирование:** карта окружения, найти стакан через CV → `(x, y, z)`
2. **Подход:** путь к стакану, остановка в 0.5 м
3. **Хватка:** статичная поза, манипуляция (передача в manipulation)
4. **Перенос:** путь к Олегу (детекция через CV `person`), скорость снижена
5. **Передача:** остановка в 0.7 м, поднять руку, дождаться приема

## Безопасность
- Если ускорение корпуса > 1.5 м/с² — замедлить (иначе кофе расплескается)
- Если препятствие ближе 0.3 м — стоп
- Если потерян баланс → G1 умеет вставать, но стакан лучше сразу опустить
