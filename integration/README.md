# Интеграция — совместные демо VOICE + COFFEE

Здесь будут end-to-end сценарии, объединяющие оба модуля.

## Сценарий 1: Голосовая команда на кофе
1. Пользователь: «принеси кофе Олегу» (микрофон)
2. ASR (распознавание речи) → текст
3. Оркестратор (coffee/llm_orchestrator) разбирает интент
4. Робот идёт за кофе, берёт, несёт
5. По пути говорит голосом Бурунова: «Олег, держи свой кофе» (voice/tts)
6. Если стакан не найден: голосом «Нихуя нет стакана, разбирайтесь»
7. Передача Олегу

## Сценарий 2: Анекдот в процессе
1. Олег: «А расскажи-ка анекдот из 86-го»
2. Voice/rag → поиск топ-1 → voice/tts (Бурунов) → проиграть
3. Пока рассказывает — не двигается (или двигается плавно)
